# bootstrap-Colors
CSV files of bootstrap color settings

This repsoitory if for the free exchange of CSV files used to color the Zen Cart bootstrap template.

We recommend that you first clone bootstrap using lat9s Clone a Template, download the starting colors and, then upload a file from here to try on your site.

All files have no guarantee of complete accessibility compliance.  If you find a problem or wish to add colors to a not-set area, please submit an issue or a PR.
